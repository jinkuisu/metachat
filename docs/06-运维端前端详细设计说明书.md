# MetaChat 重构项目 · 运维端前端详细设计说明书

## 1. 技术栈

Vue 3 + Vite + Naive UI + Pinia + vue-router + axios
独立 SPA，与客户端共用 Naive UI 组件库

## 4. 权限集成

- Sa-Token JWT 鉴权
- 路由守卫: 未登录跳 /login
- 菜单过滤: 后端返回可见菜单树, 前端动态渲染 Sidebar
- 按钮级: v-if="hasPermission('sys:user:add')"

## 5. 页面通用规范

| 元素 | 组件 | 说明 |
|------|------|------|
| 表格 | NDataTable | 分页、排序、列自定义 |
| 弹窗 | NModal | 表单编辑/详情查看 |
| 表单 | NForm+NFormItem | 统一表单验证 |
| 搜索 | NCard 内嵌 | 过滤条件行 |
| 按钮 | NButton | 新增/编辑/删除/导出 |
| 通知 | NNotification | 操作结果反馈 |

## 6. 页面设计

### 6.1 系统用户列表
  NDataTable(columns=用户名,姓名,部门,角色,状态,操作)
  + NModal 新增/编辑表单 + 角色选择器
  + 搜索: 用户名/姓名模糊

### 6.2 销售资金池
  NCard 汇总(总余额, 总冻结)
  NDataTable(columns=部门,余额,冻结,操作)
  + NModal 充值调账/冻结/解冻

### 6.3 敏感词库
  NDataTable(columns=敏感词,分类,状态,操作)
  + 批量导入: excel 文件上传
  + 搜索: 敏感词/分类
