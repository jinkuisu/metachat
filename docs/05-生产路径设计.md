# CloakBrowser 生产路径设计

## 目录结构

{app_data}/metachat/
├── cloak/                    ← CloakBrowser 内核安装目录
│   └── chrome.exe            ← CloakBrowser Chromium 内核（从 GitHub Releases 获取）
│   └── chrome.dll            ← Chromium 引擎
│   └── ...                   ← 其他 Chromium 文件
├── profiles/                 ← 账号隔离 profile 目录
├── metachat.db              ← SQLite 数据库
└── translations.db          ← 翻译缓存

## 获取方式

用户从 GitHub Releases 下载：
https://github.com/CloakHQ/CloakBrowser/releases/download/chromium-v{version}/cloakbrowser-windows-x64.zip

解压后把 chrome.exe 放到 {app_data}/metachat/cloak/ 下

## 路径发现优先级 (process.rs find_cloak_browser)

1. {app_data}/metachat/cloak/chrome.exe     ← 生产路径
2. ~/.cloakbrowser/chromium-*/chrome.exe    ← npx 安装路径
3. node_modules/cloakbrowser/              ← 开发环境
4. $CLOAK_BROWSER_PATH                      ← 环境变量

## AppConfig 配置

pub struct AppConfig {
    pub db_path: PathBuf,          // {app_data}/metachat/metachat.db
    pub cloak_dir: PathBuf,        // {app_data}/metachat/cloak/
    pub profiles_dir: PathBuf,     // {app_data}/metachat/profiles/
    pub log_level: String,
}
